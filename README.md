Low Orbit Ion Cannon rewrite in Go
how to install 
- git clone https://github.com/whitehat57/LOIC.git
- cd LOIC
- go mod init LOIC.go
- go mod tidy
- go build LOIC.go
- rm go.mod
- rm go.sum
- rm LOIC.go
- ./LOIC
